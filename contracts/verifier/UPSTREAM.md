# Vendored fork

This directory is a vendored copy of Nethermind's RISC Zero Stellar verifier,
forked unchanged for Veil's on-chain Groth16 proof verification.

- Upstream: https://github.com/NethermindEth/stellar-risc0-verifier
- Cloned: 2026-06-15
- Modifications: none (kept close to upstream, per instructions.md)

Deployed testnet addresses are tracked in `deployment.toml` and in the repo-root
`memory.md` under "Deployed artifacts".
